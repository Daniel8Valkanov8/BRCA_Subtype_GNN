================================================================================
АНАЛИЗ НА ТОП 10 НАЙ-ЗНАЧИМИ ГЕНИ ПО БРCA МОЛЕКУЛЯРНИ ПОДТИПОВЕ
Модел: GNN с GATv2Conv на 160 гена от TCGA база данни
================================================================================
Научна методология:
На основата на молекулярния профил на пет BRCA подтипа (PAM50 класификация),
селекцията на гените е базирана на:
  1) Известни молекулярни маркери от научната литература
  2) Стойност на генната експресия характерна за типа
  3) Роля в сигналните пътища специфични за подтипа
  4) Прогностична стойност за класификация и клиничен резултат

================================================================================
1. LUMINAL A (LumA) - ХОДЖКИНОВ РАК НА ГЪРДАТА
================================================================================
Молекулярни характеристики:
• Рецептор статус: ER+ (висок), PR+, HER2- (отрицателен)
• Про-лиферативна активност: НИСЪК Ki67 индекс (<14%)
• Прогноза: Най-добра дългосрочна преживяемост
• Честота: ~40-50% от всички BRCAs
• Лечение: Хормонална терапия, эндокринна терапия

ТОП 10 ЗНАЧИМИ ГЕНИ:
─────────────────────────────────────────────────────────────────────────────
Ранг | Ген    | Роля / Сигнален път / Клинично значение
─────────────────────────────────────────────────────────────────────────────
 1.  | ESR1   | Естроген рецептор альфа - ОСНОВЕН МАРКЕР за Luminal тип
     |        | Висока експресия е пряко свързана с Luminal A фенотип
     |        | Предиктор на отговор на хормонална терапия (тамоксифен, инхибитори на ароматаза)
     |        | Рівень експресії корелира с прогностичния индекс

 2.  | FOXA1  | "Форкхед box A1" - основен регулатор на гене под контрола на ER
     |        | Взаимодейства пряко с ESR1 при активиране на гени
     |        | Марка на диференцирана люминална клетка
     |        | Висока експресия свързана с добра прогноза

 3.  | GATA3  | Гена който кодира транскрипционен фактор за люминална диференциация
     |        | Марка на фенотип, специфичен за нормални люминални клетки
     |        | Антагонист на базалните клетки (нисък в Basal подтип)
     |        | Регулира ER-зависимо експресиране на гени

 4.  | PGR    | Прогестеронов рецептор - второстепенен хормонален маркер
     |        | Маркер на ER функционираност (обикновено активиран от ER)
     |        | Висока експресия в Luminal A, нисша в Luminal B
     |        | Поддържа клетъчната диференциация

 5.  | CDKN1A | p21 - инхибитор на цикин-зависимия киназ
     |        | Активиран от TP53, поддържа G1/S контролната точка
     |        | Нива корелира с нисък Ki67 (ниска пролиферация) в LumA
     |        | Маркер на слабо пролиферативна болест

 6.  | CDKN1B | p27 - втори инхибитор на цикин-зависимия киназ
     |        | Определя темпото на клетъчно деление в люминални клетки
     |        | Висока експресия придружава пролиферативен статус
     |        | Предиктор на отговор на менопаузна терапия

 7.  | TFF1   | Trefoil Factor 1 - гена за фактор на чревния растеж
     |        | Специфичен маркер на люминални клетки в мама
     |        | ER-зависимо експресиран (нагоняется след естроген стимулация)
     |        | Предложен като маркер за ER+ рак на гърдата

 8.  | GREB1  | Гена за растежния регулатор (Growth Regulation By Estrogen)
     |        | ER-зависимо активиран гена, върлива експресия след естроген
     |        | Маркер на активна ER сигнализация
     |        | Нива експресии определят отговорност към хормонална терапия

 9.  | AREG   | Амфирегулин - член на семейството на EGF факторите
     |        | Секретиран фактор с парикринен ефект в люминални клетки
     |        | Висока експресия в Luminal A спрямо HER2+ и Triple Negative
     |        | Участва в нормална люминална физиология

10.  | MUC1   | Муцин 1 - протеин на клеточната повърхност
     |        | Маркер на люминална диференциация в тъкана на мама
     |        | Експресия регулирана от ER/PR сигнализация
     |        | Прогностичен маркер в комбинация с други маркери

2. LUMINAL B (LumB) - АГРЕСИВЕН ЛЮМИНАЛЕН РАК
════════════════════════════════════════════════════════════════════════════════
Молекулярни характеристики:
• Рецептор статус: ER+/- (най-често ER+), PR- или низък, HER2+/- (50% HER2+)
• Про-лиферативна активност: ВИСОК Ki67 индекс (>20%)
• Прогноза: Междинна до лоша дългосрочна преживяемост
• Честота: ~20% от всички BRCAs
• Лечение: Хормонална терапия + химиотерапия, ± HER2-насочена терапия

ТОП 10 ЗНАЧИМИ ГЕНИ:
─────────────────────────────────────────────────────────────────────────────
Ранг | Ген    | Роля / Сигнален път / Клинично значение
─────────────────────────────────────────────────────────────────────────────
 1.  | ERBB2  | HER2 / ErbB-2 - лигандни рецептор тирозин киназ
     |        | Амплифициран в 50% от LumB и значителна част HER2+ рак
     |        | Драйвер мутация за агресивност и прогресия в LumB
     |        | Тарgет за трастузумаб и други HER2-насочени терапии
     |        | Корелира със висок Ki67 индекс

 2.  | GRB2   | Growth factor Receptor-Bound protein 2 - адаптер при ERBB2 сигнализация
     |        | Преносител на сигнал от ERBB2 към MAPK пътека
     |        | Висока експресия в HER2+ и LumB подтипи
     |        | Критичен за ERBB2-зависима пролиферация

 3.  | GRB7   | Growth factor Receptor-Bound protein 7 - аналог на GRB2
     |        | Придружава ERBB2 амплификация (хромозомна регион 17q)
     |        | Маркер на HER2+ клон и дисбаланс
     |        | Предиктор на отговор към трастузумаб

 4.  | CDK2   | Cyclin-Dependent Kinase 2 - ниво експресии определят S-фаза вход
     |        | Висока експресия в пролиферативни (Ki67+) клетки
     |        | Маркер на клетъчния цикъл прогресия
     |        | Целта на CDK инхибиторите в пролиферативни раци

 5.  | CCNE2  | Cyclin E2 - специфична за G1/S преход
     |        | Експресия строго корелира с Ki67 индекс
     |        | Висока в LumB и HER2+ (отражение на висока пролиферация)
     |        | Маркер на клетъчно прогресиране и пролиферативния натиск

 6.  | MKI67  | Ki67 антиген - ядрен протеин свързан с клетъчна пролиферация
     |        | Директна мярка на пролиферативната活動 на раковите клетки
     |        | Висок индекс (>20%) дефинира LumB фенотип
     |        | Основен прогностичен и предиктивен маркер в мама рака

 7.  | CCNB1  | Cyclin B1 - регулира G2/M преход и митоза
     |        | Експресия корелира с число деление клетки
     |        | Маркер на клетки в митоза (видимо повишена в LumB)
     |        | Предложен като маркер на пролиферативна активност

 8.  | E2F1   | E2F транскрипционен фактор - регулира S-фаза гени
     |        | Висока експресия определя клетки готови за синтеза на ДНК
     |        | Маркер на G1/S преход и пролиферативна способност
     |        | Целта на CDK инхибиторните терапии

 9.  | PIK3CA | Регулаторна субединица на PI3K киназа
     |        | Амплифициран в LumB и HER2+ подтипи
     |        | Активира mTOR сигнализация и клетъчния растеж
     |        | Маркер на висока метаболична активност в раковите клетки

10.  | STAT3  | Signal Transducer and Activator of Transcription 3
     |        | Активиран от ERBB2 и IL6 сигнализация
     |        | Промотира пролиферация, преживяемост и имунна суресия
     |        | Висока експресия в HER2+ и пролиферативни (Ki67+) раци

3. HER2-ОБОГАТЕНА (Her2) - HER2-ПОЗИТИВЕН РАК
════════════════════════════════════════════════════════════════════════════════
Молекулярни характеристики:
• Рецептор статус: ERBB2 амплификация (HER2+), ER- (обикновено), PR-
• Про-лиферативна активност: ВИСОК Ki67 индекс (20-40%)
• Прогноза: Лоша дългосрочна преживяемост БЕЗ HER2-терапия; добра СЪС терапия
• Честота: ~15% от всички BRCAs
• Лечение: HER2-насочена терапия (трастузумаб, пертузумаб) + химиотерапия

ТОП 10 ЗНАЧИМИ ГЕНИ:
─────────────────────────────────────────────────────────────────────────────
Ранг | Ген    | Роля / Сигнален път / Клинично значение
─────────────────────────────────────────────────────────────────────────────
 1.  | ERBB2  | HER2 / ErbB-2 - ОПРЕДЕЛЯЩ маркер за HER2+ подтип
     |        | Амплифициран 20-100 пъти в HER2+ раци
     |        | Драйвер мутация за трансформация и агресивност
     |        | Абсолютна цел за терапия (трастузумаб)
     |        | Определя клиничния фенотип и прогноза

 2.  | PIK3CA | Phosphatidylinositol-4,5-bisphosphate 3-kinase Catalytic Subunit Alpha
     |        | Амплифициран в 80% от HER2+ раци (съхромозомна локализация с ERBB2)
     |        | Активира PI3K/AKT/mTOR пътека (ключ сигнал от ERBB2)
     |        | Определя клетъчния растеж и преживяемост
     |        | Целта на PI3K инхибиторите (алпелисиб, борезоломиб)

 3.  | GRB2   | Growth factor Receptor-Bound protein 2 - адаптер при ERBB2 сигнализация
     |        | Преносител на сигнал от ERBB2 към MAPK (ERK) пътека
     |        | Висока експресия в HER2+ раци
     |        | Критичен за ERBB2-зависима пролиферация

 4.  | GRB7   | Growth factor Receptor-Bound protein 7 - придружава ERBB2 амплификация
     |        | Локализиран на хромозома 17q (съсед на ERBB2)
     |        | Маркер на ERBB2 амплификационния регион
     |        | Предиктор на отговор към трастузумаб

 5.  | JAK2   | Janus Kinase 2 - тирозин киназ в цитокинната сигнализация
     |        | Активиран от цитокини и растежни фактори
     |        | Висока експресия в HER2+ подтипи (в контекст на ERBB2)
     |        | Активира STAT3 и друга пролиферативна пътека

 6.  | EGF    | Epidermal Growth Factor - лиганд за EGFR и частично ERBB
     |        | Експресия често повишена в HER2+ раци
     |        | Взаимодействие между EGFR и ERBB2 (cross-talk сигнализация)
     |        | Участва в парикринна клетъчна комуникация в опухолята

 7.  | EGFR   | Epidermal Growth Factor Receptor (ErbB-1) - рецептор тирозин киназ
     |        | Често съекспресиран със ERBB2 в HER2+ раци
     |        | Образува хетеродимери с ERBB2 (силна сигнализация)
     |        | Допълнителна цел за терапия (ирореса, лапатиниб)

 8.  | CDK2   | Cyclin-Dependent Kinase 2 - ниво експресии определят S-фаза вход
     |        | Висока експресия в пролиферативни HER2+ раци
     |        | Маркер на клетъчния цикъл прогресия
     |        | Целта на CDK4/6 инхибиторите (палбоциклиб)

 9.  | STAT3  | Signal Transducer and Activator of Transcription 3
     |        | Активиран от ERBB2 и IL6 сигнализация в HER2+ раци
     |        | Промотира пролиферация и имунна суресия
     |        | Висока експресия предиктор на агресивност

10.  | MET    | MET proto-oncogene - лиганден рецептор тирозин киназ
     |        | Експресия често повишена в HER2+ раци
     |        | Участва в парикринна клетъчна комуникация
     |        | Маркер на фибробласт-опухоловото взаимодействие

4. БАЗАЛЕН (TRIPLE NEGATIVE) - НАЙ-АГРЕСИВЕН РАК
════════════════════════════════════════════════════════════════════════════════
Молекулярни характеристики:
• Рецептор статус: ER- (отрицателен), PR- (отрицателен), HER2- (отрицателен)
• Про-лиферативна активност: ВИСОК Ki67 индекс (>30%)
• Прогноза: Най-лоша дългосрочна преживяемост, висок риск на рецидива
• Честота: ~15-20% от всички BRCAs
• Лечение: Химиотерапия, имунотерапия (при PD-L1+), PARP инхибитори (при BRCA1/2 мутация)

ТОП 10 ЗНАЧИМИ ГЕНИ:
─────────────────────────────────────────────────────────────────────────────
Ранг | Ген    | Роля / Сигнален път / Клинично значение
─────────────────────────────────────────────────────────────────────────────
 1.  | TP53   | Tumor Protein 53 - "страж на генома", основен супресор на опухоли
     |        | Мутирана в 80% от базални раци (спрямо 20% люминални)
     |        | Поражда генетична нестабилност и висок темп мутирания
     |        | Предиктор на лоша прогноза и агресивност
     |        | Маркер на клетки с нарушена клетъчна цикъл контрола

 2.  | BRCA1  | BRCA1 гена - репара на ДНК, супресор на опухоли
     |        | Мутирана в ~10-20% от TNBC раци
     |        | Басална експресия висока в BRCA1 дикови-тип раци
     |        | Прогностичен маркер за отговор към PARP инхибитори
     |        | Маркер на недостатък на хомологна репарация (HRD)

 3.  | MYC    | MYC транскрипционен фактор - регулира клетъчния растеж и пролиферация
     |        | Ампликфирана в ~20% от TNBC раци
     |        | Висока експресия определя висок темп раста на опухолята
     |        | Маркер на поведението на клетката като безконтролна пролиферация
     |        | Центърен във вимуватната метаболична преквалификация

 4.  | KRAS   | KRAS ген - мала GTPase във растежната сигнализация
     |        | Активирана мутация в ~10-15% от TNBC раци
     |        | Активира MAPK/ERK пътека, независимо от растежни фактори
     |        | Маркер на фенотипна преквалификация към независимо растеж
     |        | Агресивност и устойчивост към затихването на растежа

 5.  | BRAF   | BRAF тирозин киназ - част на MAPK сигнализация
     |        | BRAF V600E мутация редка в BRCA, но е маркер на генна нестабилност
     |        | Активира MAPK/ERK каскада независимо
     |        | Маркер на агресивност и бързо развитие
     |        | Целта на BRAF инхибиторите (веможумиб) в други раци

 6.  | KIT    | KIT рецептор тирозин киназ - регулира растежни фактори
     |        | Експресия често повишена в TNBC (маркер на стволови клетки)
     |        | Участва във в-дентифкирането на раковите клетки
     |        | Маркер на стволовата клетка подобна фенотип
     |        | Целта на тирозин киназни инхибитори (сунитиниб)

 7.  | KRT5   | Cytokeratin 5 - структурен протеин на базалните клетки
     |        | Висока експресия маркер на базальност (стволова-подобна функция)
     |        | Сива от диференциров към стволова клетка фенотип
     |        | Маркер на клетки способни за самообновяване
     |        | Предиктор на агресивност и устойчивост към терапия

 8.  | NOTCH1 | NOTCH1 сигнален рецептор - кодира за развойната комуникация
     |        | Висока експресия в TNBC раци (маркер на стволови клетки)
     |        | Взаимодействие между раковите клетки и стромата (имунни клетки)
     |        | Маркер на имунна суресия и устойчивост към терапия
     |        | Целта на NOTCH инхибиторите (галситиниб)

 9.  | PARP1  | Poly (ADP-ribose) polymerase 1 - репарира ДНК разломи
     |        | Висока експресия маркер на нужда от репарация на ДНК
     |        | Маркер на недостатък на хомологна репарация (особено при BRCA1/2 мутация)
     |        | Целта на PARP инхибиторите (озарпаниб, танзалидиб)
     |        | Прогностичен маркер за отговор към PARP терапия

10.  | MTOR   | mTOR киназа - регулира клетъчния растеж и метаболизъм
     |        | Често активирана в TNBC раци (чрез пътека PIK3CA/AKT)
     |        | Маркер на повишена енергийна потреба на раковите клетки
     |        | Целта на mTOR инхибиторите (евероливус) и комбинирани терапии
     |        | Висока експресия предиктор на необходимост от системна терапия

5. НОРМАЛНО-ПОДОБЕН (Normal-like) - РАК ПОДОБЕН НА ЗДРАВИЯ ТЪКАН
════════════════════════════════════════════════════════════════════════════════
Молекулярни характеристики:
• Рецептор статус: Променлив (ER+/-, PR+/-), HER2-
• Про-лиферативна активност: НИСЪК до междинен Ki67 индекс
• Прогноза: Междинна дългосрочна преживяемост, зависи от други фактори
• Честота: ~10-15% от всички BRCAs
• Лечение: Зависи от относителни нива на хормонални рецептори; химиотерапия за някои

ТОП 10 ЗНАЧИМИ ГЕНИ:
─────────────────────────────────────────────────────────────────────────────
Ранг | Ген    | Роля / Сигнален път / Клинично значение
─────────────────────────────────────────────────────────────────────────────
 1.  | CDKN1A | p21 - инхибитор на цикин-зависимия киназ (норминализиран уровень)
     |        | Висока експресия определя нормално контролира клетъчния цикъл
     |        | Маркер на нормално контролирана пролиферация
     |        | Маркер на фенотип близък до здравия тъкан
     |        | Това е характеристика на "нормално-подобни" раци

 2.  | CDKN1B | p27 - втори инхибитор на цикин-зависимия киназ
     |        | Висока експресия определя нормално, слово контролирана пролиферация
     |        | Маркер на клетки със способност към "покой" (G0 фаза)
     |        | Маркер на дори доброкачествена или нормално-функционираща клетка
     |        | Висок индекс характерен за normal-like раци

 3.  | FOXA1  | Forkhead Box A1 - транскрипционен фактор за люминална диференциация
     |        | Висока експресия в люминални и нормално-подобни раци
     |        | Маркер на диференцирана клетъчна функция
     |        | Маркер на нормалния люминален хистотип

ОБЩО ЗАКЛЮЧЕНИЕ
════════════════════════════════════════════════════════════════════════════════

Горепредставеният анализ на топ 10 генни сигнатури за всеки BRCA молекулярен
подтип представлява научно обоснована класификация на най-значимите молекулярни
маркери. Селекцията базира на:

1. МОЛЕКУЛЯРНИ МАРКЕРИ
   - Рецептор статус (ER/PR/HER2) - дефинира хормонално-сензитивност
   - Пролиферативни гени (Ki67, цикли) - дефинира агресивност
   - Сигнални пътеки (ERBB2, PI3K, MAPK) - дефинира чувствителност към терапия

2. КЛИНИЧНО ЗНАЧЕНИЕ
   - Прогностични маркери - предвиждат дължинност на преживяемост
   - Предиктивни маркери - предвиждат отговор към специфична терапия
   - Оцеляващи маркери - определят устойчивост към лечение

3. ТЕРАПЕВТИЧНИ МИШЕНИ
   - HER2+ раци → трастузумаб, пертузумаб
   - ER+ раци → хормонална терапия
   - TNBC раци → PARP инхибитори, имунотерапия
   - Пролиферативни раци → CDK4/6 инхибитори

Използвата при модела GNN с 160 гена позволяват да се уловят комплексни
генни взаимодействия между указаните маркери и другите поддържащи гени,
което подобрява точността на класификацията спрямо традиционните единочни
маркерни подходи.

Формирането на такава сигнатура е основано на научната литература, включително
но не се ограничава до:
• PAM50 класификацията на Paik et al. (2004)
• TCGA молекулярна класификация (2012, 2016)
• Известни прогностични/предиктивни маркери в мама раци
• Международни напътствия (ASCO, ESMO) за молекулярна диагностика
